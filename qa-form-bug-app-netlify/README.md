# QA Form Bug App

A lightweight QA issue form application with reusable form presets, custom fields, Drive evidence links, Google Sheet sync, and Excel export.

This version is Netlify-ready:

- Frontend: static files in `public/`
- API: Netlify Function in `netlify/functions/api.js`
- Storage: Google Apps Script + Google Sheet + Google Drive

See `NETLIFY_DEPLOY_INSTRUCTIONS.md` for deployment steps.
