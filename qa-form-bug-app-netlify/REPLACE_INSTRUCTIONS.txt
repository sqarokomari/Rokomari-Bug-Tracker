Replace these files in your project root:

1) public/index.html
2) public/app.js
3) public/styles.css
4) netlify/functions/api.js
5) google-apps-script/Code.gs

Important:
- After replacing google-apps-script/Code.gs, open your Google Apps Script project, paste the new Code.gs content, and deploy the Web App again:
  Deploy -> Manage deployments -> Edit -> New version -> Deploy
- Then commit and push to GitHub.
- Netlify should auto-redeploy. If needed: Deploys -> Trigger deploy -> Deploy project without cache.
