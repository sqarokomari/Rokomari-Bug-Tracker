Replace these files in your project root:

1) public/index.html
2) public/app.js
3) public/styles.css
4) netlify/functions/api.js
5) google-apps-script/Code.gs

After replacing Code.gs:
- Open Google Sheet -> Extensions -> Apps Script
- Paste the updated Code.gs
- Deploy -> Manage deployments -> Edit -> New version -> Deploy

Then push to GitHub and let Netlify redeploy.
